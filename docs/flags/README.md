# 🏴 Banderas para el Selector de Idiomas

Coloca aquí los archivos PNG de las banderas.

## Archivos Necesarios:

- `chile.png` - Bandera de Chile 🇨🇱
- `uk.png` - Bandera del Reino Unido 🇬🇧
- `brazil.png` - Bandera de Brasil 🇵🇹
- `italy.png` - Bandera de Italia 🇮🇹

## Especificaciones:

- **Tamaño recomendado**: 120px × 70px
- **Formato**: PNG con transparencia
- **Fondo**: Transparente

## Nota:

El sitio actualmente usa banderas SVG integradas que funcionan perfectamente. Solo necesitas añadir archivos PNG si quieres personalizar las banderas.

Ver `BANDERAS-PNG.md` en la raíz del proyecto para más instrucciones.
